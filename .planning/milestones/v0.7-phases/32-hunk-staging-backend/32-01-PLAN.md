Replace the 3 stub functions with real implementations. All three follow the same pattern from RESEARCH.md:

**stage_hunk_inner** (replaces the stub):
1. Open repo via `open_repo_from_state(path, state_map)?`
2. Generate diff: `let mut diff_opts = git2::DiffOptions::new(); diff_opts.pathspec(file_path);`
3. `let diff = repo.diff_index_to_workdir(None, Some(&mut diff_opts))?;`
4. Check `diff.deltas().len() == 0` => return `TrunkError::new("file_not_found", format!("No unstaged changes for: {}", file_path))`
5. Count hunks via `git2::Patch::from_diff(&diff, 0)?` => `.ok_or_else(|| TrunkError::new("file_not_found", "Binary or unchanged file"))?`
6. Check `(hunk_index as usize) >= patch.num_hunks()` => return `TrunkError::new("stale_hunk_index", format!("Hunk index {} out of range (file has {} hunks)", hunk_index, num_hunks))`
7. `drop(patch);` to release borrow on diff
8. Create `ApplyOptions::new()` with `hunk_callback` that tracks a `current: usize` counter, returns `true` only when `current == hunk_index as usize`, increments `current` each call
9. `repo.apply(&diff, git2::ApplyLocation::Index, Some(&mut apply_opts)).map_err(|e| TrunkError::new("hunk_apply_failed", e.message().to_owned()))?`
10. Return `Ok(())`

**unstage_hunk_inner** (replaces the stub):
1. Open repo via `open_repo_from_state(path, state_map)?`
2. Generate REVERSED diff: `diff_opts.pathspec(file_path).reverse(true);`
3. If `is_head_unborn(&repo)`: `repo.diff_tree_to_index(None, None, Some(&mut diff_opts))?`
   Else: `let head_tree = repo.head()?.peel_to_tree()?; repo.diff_tree_to_index(Some(&head_tree), None, Some(&mut diff_opts))?`
4. Same delta count check => `"file_not_found"` with message `"No staged changes for: {file_path}"`
5. Same Patch hunk count check => `"stale_hunk_index"`
6. Same `hunk_callback` pattern
7. `repo.apply(&diff, git2::ApplyLocation::Index, Some(&mut apply_opts))` with `"hunk_apply_failed"` error mapping
8. Return `Ok(())`

**discard_hunk_inner** (replaces the stub):
1. Open repo via `open_repo_from_state(path, state_map)?`
2. Generate REVERSED diff: `diff_opts.pathspec(file_path).reverse(true);`
3. `let diff = repo.diff_index_to_workdir(None, Some(&mut diff_opts))?;`
4. Same delta count check => `"file_not_found"` with message `"No unstaged changes for: {file_path}"`
5. Same Patch hunk count check => `"stale_hunk_index"`
6. Same `hunk_callback` pattern
7. `repo.apply(&diff, git2::ApplyLocation::WorkDir, Some(&mut apply_opts))` with `"hunk_apply_failed"` error mapping
8. Return `Ok(())`

Key details:
- All functions take `(path: &str, file_path: &str, hunk_index: u32, state_map: &HashMap<String, PathBuf>) -> Result<(), TrunkError>`
- The `hunk_callback` closure captures `hunk_index as usize` as `target` and a mutable `current: usize = 0` counter
- `Patch::from_diff(&diff, 0)?` returns `Option<Patch>` -- the `?` handles the `git2::Error`, then `.ok_or_else(...)` handles the `None` case

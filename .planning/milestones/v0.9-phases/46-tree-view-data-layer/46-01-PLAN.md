    Algorithm (trie-insert then post-process, per research recommendation):

    1. **Early return:** If `files.length === 0` return `[]`.

    2. **Phase 1 -- Build intermediate trie:**
       Define a local interface for intermediate nodes:
       ```typescript
       interface IntermediateDir {
         children: Map<string, IntermediateDir>;
         files: FileStatus[];
         path: string;
       }
       ```
       Create root: `{ children: new Map(), files: [], path: '' }`.
       For each FileStatus in `files`:
       - Split `file.path` by `'/'` into segments.
       - Walk segments 0 to length-2 (directory components), creating intermediate dirs as needed.
       - Compute each intermediate dir's path as `segments.slice(0, i + 1).join('/')`.
       - Push the FileStatus into the final intermediate dir's `files` array (the last segment is the filename).

    3. **Phase 2 -- Convert to TreeNode[]:**
       Recursive `convert(dir: IntermediateDir): TreeNode[]`:
       - Create `result: TreeNode[]` array.
       - For each `[name, child]` in `dir.children`:
         - Build a `DirectoryNode` with `type: 'directory'`, `name`, `path: child.path`, `children: convert(child)`.
         - Apply `compress()` to the directory node.
         - Push to result.
       - For each `file` in `dir.files`:
         - Extract filename as `file.path.split('/').pop()!`.
         - Push `{ type: 'file', name: filename, path: file.path, file }` to result.
       - Return `sortNodes(result)`.

    4. **compress function:**
       ```typescript
       function compress(node: DirectoryNode): DirectoryNode {
         while (node.children.length === 1 && node.children[0].type === 'directory') {
           const child = node.children[0] as DirectoryNode;
           node = {
             type: 'directory',
             name: node.name + '/' + child.name,
             path: child.path,
             children: child.children,
           };
         }
         return node;
       }
       ```
       This satisfies D-04 (compress directory-only chains) and D-05 (do NOT compress when single child is a file).

    5. **sortNodes function:**
       ```typescript
       function sortNodes(nodes: TreeNode[]): TreeNode[] {
         return nodes.sort((a, b) => {
           if (a.type !== b.type) return a.type === 'directory' ? -1 : 1;
           return a.name.localeCompare(b.name, undefined, { sensitivity: 'base' });
         });
       }
       ```
       This satisfies D-06 (directories before files) and D-07 (case-insensitive alphabetical).
